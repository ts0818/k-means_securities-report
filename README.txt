注意事項
git cloneしたソースコードの内、
xbrl_table.pyに項目を追加すれば、CSV変換時にカラムを増やせる。
カラムについては、金融庁が公開してる「タクソノミ」を参照のこと。
「タクソノミ用語集」によると、
・EDINETタクソノミ
・DEIタクソノミ
・財務諸表本表タクソノミ
・内閣府令タクソノミ
などがある。
